## Packages
mathjs | Mathematical expression parsing and evaluation for the numerical methods
framer-motion | Smooth animations for UI panels and graph elements
lucide-react | Icons for the interface

## Notes
The backend provides /api/presets for pre-configured functions.
Using math.js to safely evaluate user-provided math expressions in the browser.
Using custom SVG plotting for optimal control over the step-by-step numerical method visualizations (tangent lines, cobweb diagrams, etc).
